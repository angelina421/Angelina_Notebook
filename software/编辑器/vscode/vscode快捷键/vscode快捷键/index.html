    常用 VS Code 快捷键¶

    基础编辑¶

    功能 Windows / Linux macOS
    打开命令面板 Ctrl + Shift + P Cmd + Shift + P
    打开文件 Ctrl + P Cmd + P
    多行选择（光标多点） Alt + 单击 Option + 单击
    选择所有匹配内容 Ctrl + Shift + L Cmd + Shift + L
    向上/下复制一行 Shift + Alt + ↑/↓ Shift + Option + ↑/↓
    向上/下移动一行 Alt + ↑/↓ Option + ↑/↓
    删除一行 Ctrl + Shift + K Cmd + Shift + K
    格式化代码 Shift + Alt + F Shift + Option + F

    查找与替换¶

    功能 Windows / Linux macOS
    查找 Ctrl + F Cmd + F
    查找并替换 Ctrl + H Cmd + Option + F
    查找下一个 F3 Cmd + G
    查找上一个 Shift + F3 Cmd + Shift + G

    文件与窗口管理¶

    功能 Windows / Linux macOS
    新建文件 Ctrl + N Cmd + N
    打开文件 Ctrl + O Cmd + O
    保存文件 Ctrl + S Cmd + S
    全部保存 Ctrl + K S Cmd + Option + S
    关闭文件 Ctrl + W Cmd + W
    关闭全部 Ctrl + K Ctrl + W Cmd + K Cmd + W
    分屏编辑 Ctrl + \ Cmd + \
    在编辑器间切换 Ctrl + 1 / 2 / 3... Cmd + 1 / 2 / 3...

    代码智能¶

    功能 Windows / Linux macOS
    跳转到定义 F12 F12
    查看定义（悬浮预览） Alt + F12 Option + F12
    查找引用 Shift + F12 Shift + F12
    重命名符号 F2 F2
    自动补全提示 Ctrl + Space Cmd + Space
    显示问题（错误/警告） Ctrl + Shift + M Cmd + Shift + M

    终端与调试¶

    功能 Windows / Linux macOS
    打开内置终端 Ctrl + ~ Cmd + ~
    新建终端 Ctrl + Shift + ~ Cmd + Shift + ~
    启动调试 F5 F5
    单步调试 F10 F10
    进入函数 F11 F11
    跳出函数 Shift + F11 Shift + F11
    设置/取消断点 F9 F9

    高效技巧¶

    • Ctrl + B / Cmd + B → 显示/隐藏侧边栏
    • Ctrl + Tab → 在打开的文件之间切换
    • Ctrl + Shift + [ / ] → 折叠 / 展开代码块
    • Ctrl + K Ctrl + C → 注释选中代码
    • Ctrl + K Ctrl + U → 取消注释
