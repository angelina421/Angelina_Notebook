# vim编辑器简介



## Vim 基本模式

| 模式         | 说明                           | 进入方式                    |
| ------------ | ------------------------------ | --------------------------- |
| **普通模式** | 默认模式，用于移动、删除、复制 | 启动 vim 时默认就是         |
| **插入模式** | 用于输入内容（打字）           | 按 `i` 进入                 |
| **命令模式** | 保存、退出等命令               | 先按 `Esc`，再输入 `:` 进入 |

## 如何使用vim

### 打开文件

```bash
vim helloworld.py
```

### 1. **编辑文件**

- 按下 `i`：进入插入模式（左下角会显示 `-- INSERT --`）

- 输入你想加的内容，例如：

  ```bash
  print("hello world")
  ```

------

### 2. **保存并退出**

按以下键：

```vim
Esc              # 退出插入模式
:                # 进入命令模式
wq               # 写入并退出（w = write, q = quit）
Enter            # 回车
```

也可以用：

```vim
:x
```

------

### 3. **退出不保存**

```vim
Esc
:q!
Enter
```

------

### 4. **保存不退出**

```vim
Esc
:w
Enter
```

------

### 移动光标

> 在**普通模式**下，才能进行移动。

| 操作       | 键              |
| ---------- | --------------- |
| 上下左右   | `k` `j` `h` `l` |
| 跳到行首   | `0`             |
| 跳到行尾   | `$`             |
| 跳到下一行 | `j`             |
| 跳到上一行 | `k`             |
