# Linux 软件管理

## python 环境安装

* 安装 python3：

    ```bash
    sudo apt update
    sudo apt install python3 python3-pip
    ```

* 验证 python 编译环境：

    ```bash
    python3 --version
    pip3 --version
    ```

## Qt 环境安装

* 安装 Qt 所需环境：

  ```bash
  sudo apt install qtbase5-dev qtchooser qt5-qmake qtbase5-dev-tools
  ```

* 测试 Qt 编译环境

  ```bash
  qmake --version
  ```
